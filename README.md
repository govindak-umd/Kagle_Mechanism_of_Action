## Pre-requisites

Please ensure that you have the following libraries pre-installed.

 - pytorch
 - seaborn
 - numpy
 - sklearn
 - matplotlib
 - pandas

## Requirements

 - While the programme can run on CPU, its much faster on GPU. Hence, please have your GPU/CUDA enabled.
 - Python 3.x
